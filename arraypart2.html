12. What is ceil and floor function and what is use of them.

Answer: Ceil is math object method which round Upwards to nearest interger and return same. floor is math object method which round Downwards to nearest interger and return same. We can use both methods when we need to Round any number to Upwards/Downwards interger.

13. What is round and how it is different from ceil and floor.

Answer: round is math object method which round to its nearest interger and return same. For example - If fraction value is greater than or equal to 0.5 then it round to it's Upwards nearest integer and it fraction value is less than 0.5 then it round to it's Downwards nearest integer.
